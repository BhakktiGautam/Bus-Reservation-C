# 🚌 Bus Reservation System (C)

A simple **Bus Reservation System built in C** using structures and menu-driven programming.  
This project simulates a basic ticket booking system where users can manage buses, passengers, and reservations.

It is part of my **25 Projects in C Journey** to strengthen my programming fundamentals.

---

## 🚀 Features

### Bus Management
- Add new buses
- Display all buses
- Search buses by **source and destination**
- Check **seat availability**

### Passenger Management
- Add passenger details
- Display passenger records
- Search passenger by ID
- Prevent duplicate passenger IDs

### Ticket Management
- Book bus tickets
- Display booked tickets
- Cancel tickets
- Automatic ticket record handling

### Reservation Statistics
- Total buses in system
- Total passengers
- Total tickets booked
